我並不是專業開發者，至少目前還不是。  
I'm not a professional developer — at least not yet.

<br>

比起從工程角度出發，
我更常從「實際使用者」的角度去思考問題。  

Rather than approaching things purely from engineering,
I usually think from the perspective of someone actually using the app.

<br>

我主要會參與 (Most of my contributions are related to)：

- 🌏 開源項目的翻譯與本地化 (Open source localization)
- ✨ 一些改善使用體驗的小修改 (Small UX improvements)
- 🧪 自己想到的實驗性功能 (Experimental ideas)
- 🔧 把「感覺哪裡怪怪的」變得更順手 (Tiny tweaks that make apps feel better)

<br>

我也正在學習開發，所以很多東西可能還不夠成熟、簡單，甚至有點粗糙。  

I'm still learning development,
so some of my projects may be simple, rough around the edges, or experimental.

<br>

但幾乎所有靈感，都來自我真的在使用這些東西。  

But almost every idea comes from actually using the software myself.
